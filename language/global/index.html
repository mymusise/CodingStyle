File and Directory Naming

Rules:

  • all lowercase, words connected with dash

  • camel case, without space

  • file extensions should be lowercase

  • can start with underscore or dot

Example:

{% highlight bash %} lemonless-icon.png head-meta.html CodingStyle HelloWorld.js HelloWorld.react.js _include .gitignore {% endhighlight %}
